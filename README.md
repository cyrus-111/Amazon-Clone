# Amazon Frontend Clone

 A website clone for amazon including its Frontend features. The webpage is not really functional and exactly the same as original Amazon website there are few things which are different and personalised but overall its same as the original, you won't actually find a difference unless you notice closely. The webpage is short, simple, clean and is styled using different attributes of CSS. There are small details in the header bar like the original website e.g If you hover over the icons in the header you will notice a white border formed as in the original webpage.
